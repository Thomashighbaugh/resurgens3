Linux: BASH Scripting & Provisioning

Colt45 GUI Installer

Using Zenity and BASH functions, this script features a GUI which gives the user dozens of options for installing programs on a freshly provisioned Linux desktop. Developed for personal use, due to my reinstallation of my OS every week or so as I am tinkering with my OS in hopes of developing a personal customization that is optimal for my purposes, this script differs from most other rapid installation applications in that the recipes are coded into the script and no remote server is hosting it. This simplification means that I incur no hosting costs, need not host a remote server to run Ansible and can easily reprovision my desktop environment without having to manually reinstall the programs I use regularly nor use the GNOME store's often broken installation paradigm.

Decommissioner

Shell script that creates an install script from the packages on the current system (Linux Only). Starting with a script that pulls off the list of packages installed on board, the script than creates (and backs up into an archive) an installation script that will install (most of) those packages for the purpose of reprovisioning my workstation as it was when it was freshly installed minus the hours remembering what all I had previously.

colt45 Automatic Reprovisioner

A script which installs a host of software that I find myself using on my workstations. Being that I am trying to learning the internals of the OS better and NPM being generally prone to breaking, I find myself reinstalling way more than I would like to. This predates the decommissioner state mentioned above, but I still find it useful as it includes install methods not available to the above script due to my use of Flatpak and Snap as well.
